# Safa Yildiz Terminplaner

Eine einfache Demo-Terminwebsite für GitHub Pages.

## Demo-Zugänge

- Admin: `admin` / `admin123`
- Mitarbeiter: `mitarbeiter` / `team123`
- Kunde: `kunde` / `kunde123`

## Wichtig

Diese Version speichert Daten nur im Browser (`localStorage`). Sie ist ideal zum Testen und Vorzeigen, aber noch nicht für echte vertrauliche Kundendaten geeignet. Für echte getrennte Konten auf mehreren Geräten braucht die Website später eine sichere Datenbank und Anmeldung, zum Beispiel Supabase.

## GitHub Pages

Lade `index.html`, `styles.css`, `app.js` und `.nojekyll` in ein GitHub-Repository. Aktiviere danach unter **Settings → Pages → Deploy from a branch → main / root → Save** die Website.
